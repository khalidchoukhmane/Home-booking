## Installation
1- Clone this repository.

```bach
git clone https://github.com/khalidchoukhmane/Home-booking.git
```
2- Install the dependency needed
```bach
npm install
```
3- Run your app
```bach
ng serve
```
4- Open another terminal to run json
```agsl
json-server --watch db.json
```
